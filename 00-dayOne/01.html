First Day Notes:

NOTES

  • When using Emmet, class is notated by using a "." and id is by "#".
  • HTML stands for Hyper Text Markup Language.
  • DOCTYPE states the type of our document. In this case, HTML.
  • Our HTML tag denotes our language to be English, per the lang="en".
  • The head tag houses metadata, or detailed data about our doc.
  • UTF-8 specifies character encoding. Each character (letter, nummber, even spaces) has a numeric value.
  • Viewport notes the device we are reading our document on.
  • Title reflect the name that displays on our browser tab.
  • Links allow us to connect to other files, such as our stylesheet. (CSS)
  • A style tag can help us inject CSS ino our document. *Not the best practice.*
  • The body tag houses all the other tags that we want to display onto our browser. This will be what the user sees!
